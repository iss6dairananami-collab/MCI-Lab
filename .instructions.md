# Repository instructions

- このリポジトリでは、機能一覧や機能項目を表示する場合、各項目をタッチ／クリックできるように実装してください。
- タッチ／クリックしたときに該当機能へ遷移する、またはその機能を起動する動作を提供してください。
- UI テキストは日本語で維持し、`index.html` 内にシンプルな構造とインラインスクリプトで実装する方針を守ってください。
- 既存の会話型アプリデザインに沿って、追加のナビゲーション要素や機能は自然なチャット／メニュー体験になるよう統合してください。

## 例
- 「認知症チェック」をタッチするとチェック機能を開始する。
- 「履歴を見る」をタッチすると保存された履歴画面に切り替える。
- 「家族に通知」をタッチすると通知操作を開始する。

## 期待されるプロンプト
- `機能の項目をタッチするとその機能にいくようにしてください`
- `機能メニューを追加して、タッチやクリックで移動できるようにしてください`
- `index.html の UI を強化して、各機能をタップ可能にしてください`
